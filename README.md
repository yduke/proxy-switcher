# Proxy Switcher

一个简洁的 Chrome 浏览器代理切换插件，支持一键切换无代理、系统代理和自定义代理三种模式，插件图标颜色随当前模式实时变化。

## 功能特性

- **三种代理模式**一键切换
  - 🟢 **无代理** — 直连互联网，完全绕过系统代理设置
  - 🔵 **系统代理** — 跟随操作系统的代理配置
  - 🔴 **自定义** — 使用用户自定义的代理服务器
- **图标颜色指示** — 插件图标实时反映当前代理模式，一眼可知当前状态
- **自定义代理配置** — 支持分别设置 HTTP / HTTPS 代理服务器，可配置认证信息和绕过列表
- **深色 / 浅色模式** — 自动跟随系统外观设置
- **状态持久化** — 重启浏览器后自动恢复上次的代理模式和配置

## 安装方法

由于此插件尚未上架 Chrome Web Store，需通过开发者模式手动加载。

1. 下载本仓库（`Code → Download ZIP`）并解压，或使用 `git clone`
2. 打开 Chrome，地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的 **「开发者模式」** 开关
4. 点击 **「加载已解压的扩展程序」**，选择解压后的 `proxy-switcher` 文件夹
5. 插件图标出现在工具栏后即安装成功

> **注意**：Chrome 73 起已不支持直接拖入 `.crx` 文件安装第三方插件，以上步骤是目前唯一的本地安装方式。

## 使用说明

### 切换代理模式

点击工具栏中的插件图标，在弹出界面中点击对应按钮即可立即生效。

### 配置自定义代理

1. 点击界面右上角的 **齿轮图标** 进入设置
2. 填写 HTTP / HTTPS 代理服务器的地址和端口
3. 如代理服务器需要认证，填写用户名和密码
4. 在「绕过代理」中填写不需要走代理的域名或 IP（逗号分隔）
5. 点击 **「保存设置」**，切换到「自定义」模式后生效

**绕过代理默认值：**
```
localhost, 127.0.0.1, 192.168.*.*
```

## 所需权限说明

| 权限 | 用途 |
|------|------|
| `proxy` | 读写浏览器代理设置 |
| `storage` | 保存代理模式和配置信息 |
| `<all_urls>` | 拦截代理认证请求，自动填写用户名和密码 |

## 已知限制

- 代理认证（用户名/密码）依赖 `webRequest` API，在 Manifest V3 中支持有限，部分场景下可能需要手动输入凭据
- 插件仅影响 Chrome 浏览器本身的网络请求，不影响系统其他应用
